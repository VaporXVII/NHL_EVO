# Overview
This folder contains the notebooks responsible for processing NHL game-level data through the Bronze, Silver, and Gold layers of the Medallion Architecture. 

# Purpose
- Ingest raw schedule data from the NHL API into the Bronze layer.
- Transform, validate, and cleanse the raw data within the Silver layer.
- Load analytics-ready data into the Gold layer for downstream use.


# File Description
- Schedules: contains notebooks for game schedules
- PBP_Data: contains notebooks for in-game play-by-play data
- Shift_Data: contains notebooks for in-game shift data
